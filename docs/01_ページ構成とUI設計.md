## 各ページの UI 構成

### ランディング（`/` — `page.tsx`）
参照: `../output/index.html`, `../interactive-globe/main.js`

Three.js による3D地球儀のインタラクティブ体験。全画面・Header/Footerなしの独立レイアウト。
地球儀 → 東南アジアズーム → ピン表示 → 活動写真閲覧、という一連のフローをすべてこのページ内で完結する。

#### STEP 2: ズームイン — 東南アジアへフォーカス + UI出現

**[ GO ASIA! ] クリック後のアニメーション**

カメラが東南アジアへズームすると同時に、ヘッダー・ナビゲーション等のサイトUIがフェードインで出現する。
地球儀のインタラクティブ体験から、通常のウェブサイトへシームレスに移行する演出。

```

┌─────────────────────────────────────────────┐

│ TSUBUKI SEMINAR       Nav  Social  JA/EN    │    （ズームと同期）
├─────────────────────────────────────────────┤
│                                             │
│   カメラが東南アジアに向かってズームイン       │
│   （3.5秒 easeInOutQuart）                  │
│                                             │
│        ╭─────────────────────╮              │
│        │                     │              │
│        │   東南アジア拡大     │              │
│        │                     │              │
│        │  📍 📍              │              │
│        │     📍  📍          │              │
│        │        📍           │              │

```

**同時に起きるアニメーション（タイムライン）**:

```
t=0s          t=1s          t=2s          t=3s     t=3.5s
│             │             │             │         │
▼             ▼             ▼             ▼         ▼
├─────────────────────────────────────────┤
│  カメラ: 地球全体 → 東南アジアへズーム     │
├─────────────────────────────────────────┤
│  タイトル・CTA: フェードアウト             │
├───────────┤
│           ├─────────────────────────────┤
│           │  Header: フェードイン         │
│           │  opacity 0 → 1              │
│           │  backdrop-blur 有効化        │
│           ├─────────────────────────────┤
│                                   ├─────┤
│                                   │ ピン │
│                                   │ 順次 │
│                                   │ 出現 │
│                                   ├─────┤
```

- **カメラ** (0〜3.5s): 地球全体 → 東南アジア地域にスムーズにズーム
- **タイトル・CTA** (0〜1s): 「TSUBUKI SEMINAR」「GO ASIA!」がフェードアウトで消える
- **Header** (1s〜3s): ヘッダーが `opacity: 0→1` でフェードイン、`backdrop-blur` も有効化
- **ピン** (3s〜3.5s+): ズーム完了付近で活動拠点のピンが順次出現（パルスアニメーション付き）
- **最新情報UI** (3.5s〜4s): Instagram最新投稿フィードが画面下部にフェードイン

**UI要素の状態遷移**:
| フェーズ | Header | 最新情報UI（Instagram） |
|---------|--------|----------------------|
| STEP 1（初期表示） | 非表示（`opacity: 0`, `pointer-events: none`） | 非表示 |
| STEP 2（ズーム中） | フェードイン（`opacity: 0→1`, `transition: 2s ease`） | 非表示 |
| STEP 3（ピン操作） | 完全表示（通常の固定ヘッダーとして機能） | 表示（画面下部に横スクロールフィード） |

**ピンパネル（クリック時）**:
- 各ピンはクリック可能（インタラクティブ）
- ピンクリック → パネルが表示される
- パネル内容（二重ソース）:
  - **Instagram最新投稿**: `#tsubukizemi_{国名}` タグの投稿を自動取得・表示
  - **活動アーカイブ**: microCMS (activity-photos) から国名フィルタで取得した厳選写真
  - 拠点名（例: カンボジア・プノンペン）
  - 活動概要テキスト
  - 詳細ページへのリンク（`/activities` 等）
- 他のピンをクリックすると、現在のパネルが閉じて新しいピンのパネルが開く
- ピン以外の地球部分をクリックするとパネルが閉じる

**最新情報UI（常時表示）**:
- 画面下部に常時表示されるInstagram投稿フィード
- 半透明背景 + backdrop-blur で地球儀に重なる
- 最新3〜5件を横スクロールカードで表示
- カードクリック → Instagramの該当投稿を新タブで開く
- 国名タグに関係なく、全投稿を時系列で表示

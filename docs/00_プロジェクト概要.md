# プロジェクト概要

## プロジェクト名
つぶきゼミ ウェブサイト（Tsubukizemi Web）

## 目的
武蔵野大学アントレプレナーシップ学部 津吹ゼミ（グローバルゼミ）の公式ホームページ。
ゼミの活動内容・理念・メンバーを発信し、新規ゼミ生の獲得やゼミのブランディングを行う。

## ターゲットユーザー
- ゼミへの参加を検討している学生
- ゼミの活動に関心のある学内外の関係者

## 技術スタック
| 項目 | 技術 |
|------|------|
| フレームワーク | Next.js 16 (App Router) |
| UI ライブラリ | React 19 |
| スタイリング | Tailwind CSS 4 |
| 言語 | TypeScript 5 |
| CMS | microCMS（ニュース記事） |
| 3D ビジュアル | Three.js（ランディング地球儀） |

## ベースとなるUI
`../tsubukizemi/pages/home.tsx` + `../tsubukizemi/styles/home.module.css` のUIデザインを基盤とし、
Next.js App Router + Tailwind CSS 4 に移行・リデザインして構築する。

### HOMEから引き継ぐUIの特徴
- **全画面ヒーロー**: 画像背景 + グラデーションオーバーレイ + テキスト + CTAボタン
- **カードベースのセクション**: 白背景 `border-radius: 24px` + `box-shadow` のカードに各セクションを格納
- **3カラムグリッド**: アクティビティ概要の `repeat(auto-fit, minmax(240px, 1fr))`
- **ニュースリスト**: 日付 + タイトルの縦並びリスト
- **教授プロフィールカード**: eyebrow + 名前 + サマリー + リンク
- **CTAセクション**: `linear-gradient(135deg, #111827, #1e3a8a)` の濃紺グラデーション背景
- **ボタンスタイル**: pill型（`border-radius: 999px`）、ホバーで `translateY(-1px)`

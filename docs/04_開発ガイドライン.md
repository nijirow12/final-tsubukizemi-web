# 開発ガイドライン

## ディレクトリ構成

```
src/
├── app/                        # App Router ページ
│   ├── layout.tsx              # ルートレイアウト
│   ├── globals.css             # Tailwind グローバルスタイル
│   ├── page.tsx                # ランディング（Three.js）
│   ├── home/page.tsx           # ホーム
│   ├── about/page.tsx          # ゼミについて
│   ├── activities/page.tsx     # 活動内容
│   ├── members/page.tsx        # メンバー
│   ├── news/page.tsx           # ニュース一覧
│   ├── news/[id]/page.tsx      # ニュース詳細
│   └── contact/page.tsx        # お問い合わせ
├── components/
│   ├── common/                 # Header, Footer, Navigation
│   ├── ui/                     # Button, Card, SectionTitle
│   ├── sections/               # HeroSection, ActivityCard, MemberCard 等
│   └── three/                  # GlobeScene, GlobeCanvas（Three.js 3D関連）
├── lib/                        # microcms.ts, utils
├── data/                       # professor.ts, members.ts
└── types/                      # TypeScript 型定義
```

## 検証コードからの移行方針

| 参照元 | 移行内容 |
|--------|---------|
| `../tsubukizemi/pages/home.tsx` | **UIの主要ベース** → Tailwind に変換して App Router page に |
| `../tsubukizemi/styles/home.module.css` | **スタイルの主要ベース** → Tailwind ユーティリティクラスに変換 |
| `../tsubukizemi/components/Layout.tsx` | Header / Footer に分割、App Router の layout.tsx で使用 |
| `../tsubukizemi/styles/layout.module.css` | Header / Footer の Tailwind 変換 |
| `../tsubukizemi/styles/globals.css` | globals.css に統合 |
| `../tsubukizemi/data/` | そのまま移行 |
| `../tsubukizemi/lib/microcms.ts` | そのまま移行 |
| `../interactive-globe/main.js` | React コンポーネント化（`components/three/`）、"use client" |
| `../output/index.html` | ランディング演出ロジックを `app/page.tsx` に移植 |
| `../tsubukiphoto/` | `public/images/` にコピー |

## レイアウト設計

```tsx
// src/app/layout.tsx の概要
// - ランディング（/）ではHeader/Footerを非表示
// - それ以外のページではHeader/Footerを表示

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        <ConditionalLayout>{children}</ConditionalLayout>
      </body>
    </html>
  )
}
```

## 命名規則
- コンポーネント: PascalCase（例: `HeroSection.tsx`）
- ユーティリティ: camelCase（例: `formatDate.ts`）
- ディレクトリ: kebab-case（例: `hero-section/`）
- スタイリング: Tailwind CSS ユーティリティクラス（CSS Modules は使用しない）

## 開発コマンド
```bash
npm run dev    # 開発サーバー起動
npm run build  # ビルド
npm run start  # プロダクションサーバー起動
npm run lint   # リント実行
```

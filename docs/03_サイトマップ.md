# サイトマップ

## ページ階層構造

```
/ (ランディング - Three.js 地球儀)
│
├── /home (ホーム - ゼミの全体像)
│
├── /about (ゼミについて - 理念・教授紹介)
│
├── /activities (活動内容)
│   ├── 海外での挑戦
│   ├── クラウドファンディング
│   └── Challenger訪問
│
├── /members (メンバー)
│   ├── 教授プロフィール
│   └── 学生一覧
│
├── /news (ニュース一覧 - microCMS)
│   └── /news/[id] (ニュース詳細)
│
└── /contact (お問い合わせ)
```

## ページ遷移

```
[ランディング /]
    │
    │  GO ASIA! (ズームアニメーション)
    ▼
[ホーム /home] ←──→ ナビゲーションで全ページ相互遷移
    │
    ├──→ [ゼミについて /about]
    ├──→ [活動内容 /activities]
    ├──→ [メンバー /members]
    ├──→ [ニュース /news] ──→ [記事詳細 /news/[id]]
    └──→ [お問い合わせ /contact]
```

## ナビゲーション構成

ヘッダーナビに表示するリンク:
1. Home
2. About
3. Activities
4. Members
5. News
6. Contact

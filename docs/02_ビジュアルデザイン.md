# ビジュアルデザイン方針

## デザインコンセプト
HOMEページのUIをベースに、「グローバル × 挑戦」を体現するモダンでクリーンなデザイン。
白背景のカードセクション + 濃紺アクセントの統一感。Three.js地球儀ランディングでインパクトを出す。

## アニメーション・インタラクション（HOMEから継承）

### ボタン
| 状態 | スタイル |
|------|---------|
| デフォルト | `shadow-[0_16px_34px_rgba(15,23,42,0.32)]` |
| ホバー | `hover:-translate-y-px hover:shadow-[0_20px_40px_rgba(15,23,42,0.36)]` |
| トランジション | `transition-[transform,box-shadow] duration-200 ease-out` |

### ナビリンク
- `::after` 擬似要素で下線
- `scaleX(0)` → ホバー時 `scaleX(1)`、`transition: 0.2s ease`

### カード
- ホバー: `translateY(-2px)` + シャドウ拡大

### ランディング演出
1. テキストフェードイン
2. 地球儀自動回転
3. CTA → アジアへズーム（3.5秒 `easeInOutQuart`）
4. 白フェードアウト → `/home` へ遷移
